Intro to Advanced Joystick Configs - Part 2

2019-01-11 Written by ventorvar
#Star Citizen  #JoystickGremlin  #HOTAS  #HOSAS 

Note: This is part 2 of a series. If you haven’t already installed all the required tools, go check out part 1 first.

  • Open JoystickGremlin, load profile and ensure correct devices are set

  • Ensure JoystickGremlin is active (green controller icon) and check Monitor vJoy that input is affecting the vjoy controller

  • Delete USER folder

  • Launch StarCitizen

  • Copy Actions.xml to USER/Control/Mappings (restart?)

  • Options -> Keybindings -> Advanced -> Load Defaults -> HOSASAT

    • select keyboard, mouse and joystick
